# Changelog

## 0.1.0

- Initial release
- Automatic language detection from TTS text
- Dynamic voice selection based on detected language
- Transparent STT proxy forwarding
- Configurable language→voice mappings via HA UI
- Support for English, Spanish, Italian, Portuguese, French, and German
- Single Piper instance serves multiple voices
- Bring Your Own Whisper/Piper (BYOW) architecture
